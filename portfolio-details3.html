Re-design of CSN Sign Language website

Project information

    • Category: UX/UI Design
    • Client: CSN (Swedish Board of Student Finance)
    • Course: User Experience and Interaction
    • Project Date: Winter 2024
    • Tools Used: Figma, User Research Methods

    Project Overview

    This project involved reimagining the Sign Language section of CSN's website, which provides crucial information about student loans in Sweden. Our goal was to transform a cluttered, video-heavy page into an intuitive, user-friendly interface that better serves the deaf and hard-of-hearing community.

    Design Solutions

    • Implemented a categorized, grid-based layout for easy content scanning
    • Introduced intuitive icons to represent different information categories
    • Created a responsive design to ensure accessibility across devices
    • Developed a clear hierarchy of information to guide users
    • Integrated user-friendly video players with additional controls
